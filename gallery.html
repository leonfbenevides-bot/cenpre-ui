Referência de props

Contrato de cada componente para o dev. Estados interativos: default · hover · active · focus-visible · disabled. Todos aceitam className (merge via tailwind-merge).

Button

Prop Tipo Default Descrição
variant primary | secondary | ghost | link primary Estilo visual.
size xs | sm | md | lg md Altura: 32 / 36 / 40 / 44px.
fullWidth boolean false Ocupa 100%.
leftIcon / rightIcon ReactNode — Ícone (Lucide) antes/depois.
asChild boolean false Renderiza o filho (ex.: <a>) com o estilo.

Tag · Input · Textarea · Avatar · IconChip

Componente Props Notas
Tag tone (neutral|brand|accent), size (sm|md) brand = CENPRE; accent = parceiros.
Input / Textarea label, hint, error, leftIcon, …props Acessível (label, aria-invalid, aria-describedby).
Avatar src, fallback, icon, size Logo da empresa → iniciais → ícone default (com onError).
IconChip size, tone Quadrado com ícone 24px.

Cards · JobCard · NewsCard

Componente Props Uso
Card as, padding, interactive Container base (branco, borda ash-300, radius 16).
FeatureCard icon, title, description Módulos e "CENPRE em números".
StepCard number, title, description, icon? Passos numerados.
DocCard label, icon, items[] Documentos por etapa.
JobCard title, company, area, source, modality, location, salary, logoSrc, actionLabel, href Card de vaga.
NewsCard image, author, date, title, excerpt, tags[], href Notícia/artigo (capa + meta + tags).

Interativos · Carousel · HeroBanner

Componente Props / API Notas
Accordion composável ou AccordionList items={[{question, answer}]} Radix (teclado + ARIA), animação de altura.
Tabs composável ou TabsPills items={[{value, label, content}]} Radix; pílula ativa magenta.
Carousel slides[], ariaLabel, arrows, dots Setas + dots + teclado ← →, loop.
HeroBanner image, brand, title, description, actions Hero escuro; roda dentro do Carousel.
Pagination / LogoBand page, totalPages, onPageChange / items[] Paginação e faixa de parceiros.
