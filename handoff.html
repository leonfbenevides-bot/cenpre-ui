Primitivos

Button

Botão base com variants (CVA). Suporta asChild para virar link mantendo o estilo.

Primitivo
Prop Tipo Default Descrição
variant "primary" | "secondary" | "ghost" | "link" "primary" Estilo visual.
size "sm" | "md" | "lg" "md" Tamanho.
fullWidth boolean false Ocupa 100% da largura.
leftIcon / rightIcon ReactNode — Ícone antes/depois do texto.
asChild boolean false Renderiza o filho (ex.: <a>) com o estilo do botão.
Estados
defaulthoveractivefocus-visibledisabled
Uso
<Button variant="primary" size="md">Enviar contato</Button>
<Button asChild><a href="/vagas">Ver vagas</a></Button>

Tag

Pílula/badge. Usada em cards de vaga (área, fonte, modalidade) e eyebrows.

Primitivo
Prop Tipo Default Descrição
tone "neutral" | "brand" | "accent" "neutral" brand = CENPRE (magenta); accent = parceiros (CIEE/NUBE).
size "sm" | "md" "md" Tamanho.
Uso
<Tag>Marketing</Tag>  <Tag tone="brand">CENPRE</Tag>  <Tag tone="accent">NUBE</Tag>

Composição

Card · FeatureCard · StepCard · DocCard

Família de cards sobre o Card base (fundo ash-100, borda ash-300, radius 16).

Composição
Componente Props Uso
Card as, padding, interactive Container base.
FeatureCard icon, title, description Módulos da plataforma e "O CENPRE em números".
StepCard number, title, description, icon? Passos numerados (badge magenta).
DocCard label, icon, items[] Documentos por etapa (Início/Durante/Encerramento).

JobCard

Card de vaga com avatar da empresa, tags, localização, bolsa e ação. Serve em grid ou lista.

Composição
Prop Tipo Descrição
title / company string Cargo e empresa.
area / source / modality string Tags: área, fonte (CENPRE/NUBE), modalidade.
location / salary string Localização e bolsa.
logoSrc string Logo da empresa (Avatar).
actionLabel / href / onAction string / () => void Botão (default "Ver vaga" → plataforma). 
<JobCard area="Marketing" source="CENPRE" modality="Estágio"
  title="Assistente de Marketing" company="Tech Solutions"
  location="Campos dos Goytacazes, RJ" salary="R$ 1.000,00 + Benefícios" />

Convenções shadcn/ui: variants tipadas com CVA · acessibilidade dos interativos via Radix · cn() (clsx + tailwind-merge) para merge de classes · componentes são fonte no repositório (copy-in) — o time adapta livremente.
